# 🎓 Meus Certificados de Cursos

[![Licença: MIT](https://img.shields.io/badge/Licen%C3%A7a-MIT-blue.svg)](LICENSE)

Bem-vindo ao meu repositório de certificados! Aqui guardo todos os certificados que obtive ao concluir cursos, workshops, treinamentos, etc. O objetivo é manter um histórico organizado das minhas qualificações e compartilhá-los de forma transparente.

## 📂 Estrutura do Repositório

Os certificados estão organizados por ano de conclusão do curso com o nome do curso e a plataforma onde o curso foi feito:

````
/Certificados
├── /2025 --> Cursos concluidos em 2025
├── /2026 --> Cursos concluidos em 2026
````

## 🔍 Como visualizar

1. Navegue pelas pastas acima.
2. Clique na pasta desejado.
3. Selecione um arquivo.
4. O GitHub exibirá uma pré-visualização (para PDFs pode ser necessário baixar ou usar um plugin do navegador).

## ✅ Validação

Todos os certificados aqui contêm:
- Nome do curso
- Carga horária (quando disponível)
- Assinatura digital ou carimbo da instituição
